# Book-Search-Engine

## Description

This project is to disply our ability to create a GraphQL API built with Apollo Server. The app will be built using the MERN stack, MongoDB database, Node.js/Express.js server and API, with a React front end.

## Installation
    
  Follow these steps to install or run this application:

 Step 1: Clone the code <br>
 Step 2: Locate the cloned code via terminal <br>
 Step 3: Run "npm install" <br>
 Step 4: Run "npm start". <br>
